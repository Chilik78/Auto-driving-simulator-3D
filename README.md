# Клавиши управления транспортом игрока/окружением в симуляции Unity:
# "Mouse" - управление игровой камерой
# "WASD" - управление транспортом
# "F" - включение/выключение фар транспортом
# "Q/E" - включение/выключение поворотных огней транспорта
# "Space" - ручной тормоз транспорта
# "L" - включение/выключение дорожных фонарей (для оптимизации симуляции)
# "V" - включение/выключение мигалок у грузовика
# "G" - выход в главное меню выбора транспорта
# "Esc" - отображение курсора мыши (для выхода из режима управления камерой)

# Система игровых штрафов
# 1. Езда с выключенными фарами (-5 очков)
# 2. ДТП с участием другого транспорта (-30 очков)
# Изначально игрок начинает движение с суммой в 100 очков.
# При достижении 0 очков игрока перемещают в главное меню выбора транспорта.
# При получении штрафов у игрока динамически списывают очки.

# Возможные ошибки/проблемы при запуске симуляции в Unity
# 1. При нажатии кнопки "Поехали" в главном меню выбора транспорта Unity зависает.
# Решение: 1.1 Перезапустите проект Unity без последующего включения симуляции
#          1.2 В интерфейсе Unity "Hierarchy" нажмите "MyScripst" -> В правой части проекта Unity отобразится окно "Inspector".
#          1.3 Раскройте окно скрипта "Main Handler".
#          1.4 В поле Sumo Path введите свой путь до папки SUMO - "C:\....\Sumo".
#          1.5 Запустите симуляцию
# 2. Низкая частота кадров в симуляции (низкий FPS)
# Решение: 2.1 Выключите освещение дорожных фонарей путём нажатия на клавишу "L".
#          Если выключение городского освещение не помогло, то
#          2.2 Выключите симуляцию -> 1.2 -> 1.3.
#          2.3 В поле "Max Num Veh" установите значение меньше данного установлено. 
#          (Параметр "Max Num Veh" определяет максимальное число генерируемого транспорта в симуляции) 
#          2.4 Запустите симуляцию.
